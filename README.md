# Colors For You

A modern web application that generates custom color palettes with beautiful animations and interactive UI, built with Next.js.

## Features

- **Custom Color Palettes**: Generate color combinations based on your preferences
- **Image-based Palettes**: Extract colors from your favorite images
- **Interactive UI**: Smooth animations and responsive design powered by Framer Motion
- **Dark/Light Mode**: Toggle between themes for comfortable viewing
- **Save Favorites**: Store your preferred color combinations
- **Multiple Tools**: Access various color tools including palette creator and more

## Technologies Used

- **Next.js 15**: React framework with App Router
- **React 19**: For building the user interface 
- **Framer Motion**: Powering smooth animations and transitions
- **Firebase**: Backend hosting and deployment
- **Tailwind CSS**: Utility-first CSS framework

## Pages

- **Home**: Main landing page with interactive features
- **Creator**: Custom color palette creation tool
- **Colors**: Browse and explore color combinations
- **Image Palette**: Extract color palettes from images
- **About**: Information about the application
- **Contacts**: Get in touch with the team

## Getting Started

### Prerequisites

- Node.js (v14 or later)
- npm or yarn

### Installation

1. Clone the repository
   ```
   git clone https://github.com/YOUR-USERNAME/colorsforyou.git
   cd colorsforyou
   ```

2. Install dependencies
   ```
   npm install
   # or
   yarn install
   ```

3. Run the development server
   ```
   npm run dev
   # or
   yarn dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the application

## Deployment

The application is configured for Firebase hosting deployment:

```
npm run deploy
```

This command builds the application and deploys it to Firebase hosting.

## UI Features

- **Smooth Scrolling**: Fluid scroll effects with parallax animations
- **Interactive Cursor Effects**: Custom star cursor animations
- **Dynamic Borders**: Responsive dotted borders that adapt to the theme
- **Animated Components**: Spring animations for UI elements
